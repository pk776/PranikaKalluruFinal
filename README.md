## 1. Project Overview  
This project demonstrates the development of a fully functional, multi‑page website using foundational web technologies. The site includes a **Home Page**, **Portfolio**, **Design Style Guide**, and **Client Site**, each designed to showcase different aspects of front‑end development, design principles, and user‑centered structure.

The project follows academic expectations for IS117 by incorporating semantic HTML, external CSS, responsive layouts, accessibility considerations, and organized file structure.

**Live Website:**  
https://pk776.github.io/PranikaKalluruFinal/

## 2. Project Objectives  
This project was developed to meet the following academic objectives:

- Demonstrate proficiency in HTML5 structure

## 6. Design Principles Applied  

### Swiss (International Typographic) Style  
This project incorporates several Swiss Design principles, including:

- Grid‑based layout  
- Asymmetric composition  
- Sans‑serif typography  
- High contrast and visual clarity  
- Minimalist color palette  

These principles support readability, structure, and user‑centered design.

---

## 7. Accessibility Considerations  
The site includes:

- Semantic HTML elements  
- Alt text for images  
- Clear link labels  
- Consistent heading hierarchy  
- High‑contrast text  

These choices improve usability for all users and align with WCAG guidelines.
