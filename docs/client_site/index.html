CASE STUDY

Client Site Redesign

A polished, professional website for a fictional counseling center, transforming a dense, dated layout into a calm, modern, and welcoming experience. This project focuses on clarity, hierarchy, and an accessible, responsive interface that feels trustworthy and human.

Role: Designer · Developer Tools: Figma, VS Code, HTML, CSS Timeline: 2 weeks Deliverables: Homepage, Brand System, UI Components

OVERVIEW

Creating a Calmer First Impression

The original site was text-heavy, visually noisy, and difficult to scan. The redesigned experience introduces a calmer visual language, clear entry points, and focused messaging so visitors can quickly understand who the client is, what they offer, and how to get started.

SERVICES

How I Supported the Client

The redesign touched multiple layers of the experience, from visual identity to content structure and UI components that make future updates easier.

Brand Identity

Refined typography, a balanced color palette, and consistent layout patterns to build trust.

Responsive Web Design

Layouts that adapt gracefully from mobile to desktop, keeping key actions always visible.

Content Restructure

Rewriting and reorganizing content for clarity, readability, and emotional connection.

UI Components

Reusable sections, buttons, and patterns that support a scalable, consistent interface.

PROCESS

How I Approached the Redesign

I treated this as a full case study: understanding the existing experience, defining priorities, exploring layout options, and then building a responsive, accessible implementation.

1. Discovery

Analyzed the existing site to identify pain points in hierarchy, color usage, and content density.

2. Wireframing

Explored layouts that prioritized clarity, with clear sections for welcome, services, and getting started.

3. Visual Design

Introduced a calmer palette, modern type choices, and generous spacing to support accessibility.

4. Development

Built the layout using semantic HTML and modular CSS, ensuring it adapts smoothly across screen sizes.

FEEDBACK

REFLECTION

What I Learned

This project reinforced how much small shifts in typography, spacing, and wording can change how a brand feels. If I extended this further, I’d design a resource hub and a guided onboarding flow that walks new clients from first visit to first appointment with even more clarity and care.
