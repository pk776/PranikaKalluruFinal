Project Three

A branding exploration using grid systems and typographic hierarchy.

Overview

This project explores branding through the lens of Swiss Design. The focus was on consistency, alignment, and typographic voice.

Process

I experimented with grid‑based logo sketches, modular layouts, and a limited color palette to create a cohesive identity.

Outcome

The final branding system is clean, structured, and visually unified, with strong emphasis on clarity and balance.
