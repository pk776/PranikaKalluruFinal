Project One

A clean, structured layout demonstrating Swiss design principles.

Overview

This project explores the foundations of Swiss Design: grid systems, spacing, and typographic clarity. The goal was to create a layout that feels balanced, intentional, and visually calm.

Process

I used a strict grid, consistent spacing scale, and a minimal color palette. The design evolved through iterations focused on alignment and hierarchy.

Outcome

The final result is a clean, modern layout that reflects Swiss principles while remaining functional and readable.
