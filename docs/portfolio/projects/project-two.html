Overview

This project explored teamwork, version control, and collaborative web development. Each team member contributed a unique section to a unified themed website, requiring coordination, communication, and consistent design execution.

Process

  • Designed and coded my assigned section
  • Collaborated through GitHub branching and pull requests
  • Integrated multiple design styles into one cohesive theme
  • Ensured consistency in typography, spacing, and layout
  • Resolved merge conflicts and maintained clean version history

Outcome

The final site demonstrated strong collaboration, technical coordination, and the ability to merge multiple design perspectives into a unified experience. The project strengthened teamwork, GitHub workflow skills, and design alignment.
